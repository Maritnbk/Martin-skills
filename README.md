# Martin-skills

**个人专属 AI 技能包 — Personal AI Skill Pack**

本仓库是为 AI 编程助手（OpenCode / Claude Code  等）编写的自定义技能集合。每个技能封装了一个高频、重复且容易出错的真实工作场景，通过结构化的核对流程+自动化脚本，帮助 AI 助手在该场景下稳定交付高质量结果。



## 技能一览

| 技能                                                      | 说明                                      |
| ------------------------------------------------------- | --------------------------------------- |
| [audit-report-confirmation](#audit-report-confirmation) | 审计报告核对 — 系统化校对审计报告与汇总表的财务数据、内部勾稽关系及文本质量 |



## 使用方式



如果你使用 **OpenCode 等编程助手**，在对话中直接说：

```
安装这个 skill：https://github.com/Maritnbk/Martin-skills/tree/master/audit-report-confirmation
```

AI 会自动从 GitHub 下载并放到正确的位置。每个 skill 文件夹都可以单独安装。
