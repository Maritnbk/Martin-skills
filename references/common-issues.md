# 常见错别字模式与陷阱汇总

本文件收集了审计报告核对实践中发现的真实文本问题，按类别组织。**建议在执行核对三时逐一对照这些模式检查报告文本。**

> **这些模式来自真实案例。** 每条都在实际审计报告中出现过，不是理论推导。把这些模式记住，能大幅提高错别字检查效率。

---

## 一、缺字

最常见的问题类型，通常是录入或 OCR 识别过程中漏字。

| 错误原文 | 应为 | 说明 |
|---------|------|------|
| 政府拨情况 | 政府拨款情况 | 缺"款"字 |
| 未政府补助收入 | 未收到政府补助收入 | 缺"收到" |
| 所得费收入 | 所得税费用收入 | 缺"税"或表述不完整 |
| 银行款 | 银行存款 | 缺"存" |
| 计入当损益 | 计入当期损益 | 缺"期" |

**检查技巧：** 重点关注"的""了""款""收""费""税""期"这些高频字的缺失。

---

## 二、错字

用字不当，通常是音近或形近字混淆。

| 错误原文 | 应为 | 说明 |
|---------|------|------|
| 认证严格执行 | 认真严格执行 | "认证"→"认真"，音近 |
| 关与 | 关于 | "关与"→"关于"，音近 |
| 令一方面 | 另一方面 | "令"→"另"，音近 |
| 既使 | 即使 | "既"→"即"，音近 |
| 做为 | 作为 | "做"→"作"，形近/音近 |
| 截止目前 | 截至目前 | "截止"→"截至"，用词不当 |

---

## 三、标点符号问题

| 问题 | 示例 | 正确写法 |
|------|------|---------|
| 重复句号 | "。。" | "。" |
| 英文标点混入 | "管理费用530,831.78。"中的句号是英文 | 使用中文标点"。" |
| 括号不配对 | 中文内容用了英文括号")" | 使用中文括号"）" |
| 书名号间缺标点 | 《章程》《财务制度》 | 《章程》、《财务制度》 |
| 引号不配对 | 前引号和后引号数量不一致 | 成对使用 |

---

## 四、数字与格式问题

| 问题 | 错误示例 | 正确示例 |
|------|---------|---------|
| 数据后缺"元" | "管理费用530,831.78" | "管理费用530,831.78元" |
| 逗号位置错误 | "190,0000" | "1,900,000" |
| 年度格式不一致 | "2024年"和"24年"混用 | 统一用"2024年" |
| 多余空格 | "收入合计  1,000,000" | "收入合计 1,000,000" |
| 表格对齐错乱 | 某列数字与其他列错位 | 逐列检查对齐 |

---

## 五、表述不通顺

| 问题 | 示例 |
|------|------|
| 缺主语 | "经审计，认为财务报表公允反映了..." → 缺"我们"或"注册会计师" |
| 动宾不搭配 | "进行了制度的完善和修订了流程" → 可改为"完善了制度，修订了流程" |
| 语序混乱 | "对2024年度协会的收支情况进行了审计" → 应调整语序 |
| 赘余重复 | "报告期内，报告期内，协会共完成..." → 重复 |
| 模板残留 | 未替换的占位符如"【协会名称】"等 |

---

## 六、汇总表特有陷阱

| 陷阱 | 说明 |
|------|------|
| 多余表头字符 | 如"级：合计"应为"合计" |
| 公式残留 | xlsx 单元格显示为公式而非数值 |
| 合并单元格 | 合并单元格在 Markdown 中可能只出现一次值 |
| Sheet 命名不规范 | 如默认 Sheet 名未修改 |
| 空行/空列干扰 | 提取结果中出现额外空行导致表格错位 |

---

## 七、数据常见陷阱

| 陷阱 | 说明 | 处理方式 |
|------|------|---------|
| 浮点数精度 | xlsx 中 `5102.910000000001` | 四舍五入到两位小数 |
| 公式计算误差 | 合计列是 SUM 公式，可能有舍入误差 | 手动重算验证 |
| 千分位逗号 | 报告中 `1,775,477.94` | 去掉逗号做数值比较 |
| 多层表头 | 非限定性/限定性分列 | 逐列对齐，不要跳列 |
| 多 Sheet | 汇总表有收支表+明细表 | 所有 Sheet 都要提取分析 |
| 协会名简称 | 报告中可能用简称 | 统一使用全称做对照 |
